ANIGMA

転送・データ移行専用暗号化システム

システム概要

Anigmaは転送・データ移行専用に設計された次世代暗号化システムです。 ユーザーはPINとワンタイムパスワードを一度認証するだけで、一時的な転送と移行が可能になります。 暗号化と復号化は、AES-256-GCMを使用し、鍵にはユーザーの入力するPIN（数字のみのパスワード）を使用します。 システムにはクラウドを使わず、オンデバイスのみでも完結できるように設計されています。

特徴

• 転送特化暗号化エンジン — AES-256-GCMによる高速かつ安全な処理

• PIN + TOTP 認証 — 一時的な安全セッションを即時生成

• 一時鍵生成方式 — ユーザーPINから安全な鍵を導出

ステータス

現在、Anigmaは開発中です。転送・データ移行専用の暗号化プロトコルを実装中です。詳細な情報は準備が整い次第公開されます。

Anigmaは、転送や移行などを目的に作成された暗号プロトコルです。このプロトコルは、

status.properties
// システムステータス
STATUS = IN_DEVELOPMENT
VERSION = 0.1.0-alpha
TYPE = TRANSFER_MIGRATION_ENCRYPTION
PROTOCOL = ANIGMA_V1
